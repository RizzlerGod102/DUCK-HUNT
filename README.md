🦆 Duck Hunt Game – Rules

Objective:
Shoot as many ducks as you can before they escape!

How to Play:

Download the game ZIP file and extract it to your computer by pressing the green code button

Open index.html in your browser (works on Chrome, Firefox, or Edge).

Ducks 🦆 will appear on the screen and move around.

Click on a duck 🖱️ using your mouse to shoot it and earn points 🏆.

Each duck you shoot increases your score by 1.

When all ducks are shot, a dog 🐶 will appear to celebrate your victory.

After 5 seconds, a new set of ducks will appear automatically.
